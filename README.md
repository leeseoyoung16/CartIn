# 📘 쇼핑몰 프로젝트

Spring Boot 기반으로 개발된 **쇼핑몰 프로젝트**입니다.  

---

## ⚙️ 기술 스택

- Java 17
- Spring Boot 3.5.3
- Spring Web / Spring Data JPA
- JWT / Spring Security
- Lombok / Validation
- MySQL
- Postman (API 테스트용)
- JUnit 5 / AssertJ / Spring Boot Test (테스트 코드 작성용)

---

## 🛠️ 진행 상황

| 기능 항목 | 상태 | 비고 |
|-----------|------|------|
| 사용자 인증 | ✅ | 회원가입, 로그인 기능, jwt 인증 |
|  |  |  |

## 🔗 주요 API 명세

### 🔐 사용자 인증
| 메서드 | URI            | 설명             |
|--------|----------------|------------------|
